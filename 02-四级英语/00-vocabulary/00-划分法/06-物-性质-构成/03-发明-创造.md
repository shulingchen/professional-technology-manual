## invent
```
/ɪn'vent/

v. 发明, 创造, 编造, 捏造

What excuse did he invent this time?
他这次编了什么借口？
```

## construct
```
/kən'strʌkt/
v. 构造，建造；构想

n. 构想；建造物

When was the bridge constructed?
那座桥是何时修建的？
```

## devise
```
/dɪ'vaɪz/
v. 设计; 发明; 策划; 想出; [律]遗赠;
n. 遗赠财产的遗嘱(或其中的条款), 遗赠的财产

The team's task is to devise a method to improve efficiency and reduce costs.
这个团队的任务是构想出增效减支的方法。

deviate 背离
derive 源自
device 设备
```
