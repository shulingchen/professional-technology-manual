## procession
```
/prə'seʃn/

n. 行列, 列队行进, 游行, （一个接着一个而来的）一队人

v. 排队前进, 沿着(街道)列队行进

The long wedding procession follows the bride and bridegroom.

长长的婚礼队列跟在新郎和新娘身后。
```

## process
```
n. 进程, 流程, 工序, 工艺流程
v. 处理, 加工, 审核, 数据处理
adj. 经过特殊加工的, (用化学方法等)处理过的, 照相制版的, 三色版的

The process of evolution is usually long and gradual.
进化的过程往往是漫长且循序渐进的。
```

## parade
```
/pə'reɪd/
v. 游行; 列队行进; 炫耀; 展示
n. (庆祝)游行; 队列; 阅兵行进; 炫耀; 展示

The circus performers and animals paraded down Main Street.
马戏团的演员们和那些动物沿着主街游行。
```
## array
```
n. 数组；阵列；一大批

v. 排列；布置；配置（兵力）

例句

He was unable to escape the array of facts.

他无法躲避一连串的事实。
```
