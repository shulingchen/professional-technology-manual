## crucial
```
/'kruːʃl/

adj. 至关重要的, 关键性的

The reform of the healthcare system became a topic of crucial importance in the presidential election debate.

医疗系统改革成了总统大选辩论中的至关重要的话题。
```

## vital
```
/'vaɪtl/
adj. 必不可少的；重要的；生机勃勃的，充满生机的

The police play a vital role in our society.
警察在我们的社会中起着极其重要的作用。
```
## imperative
```
n. 重要紧急的事, 必要的事；命令, 规则；祈使语气, 祈使语气动词

adj. 重要紧急的, 迫切的；必须服从的, 强制的；命令的, 祈使的
Improving the country's economy is a political imperative for the new president.

改善国家经济是新总统施政的首要任务。
```
## decisive
```
adj. 决定性的，关键的；坚决的，果断的

例句

The government must take decisive action on gun control.

政府必须在枪支管制方面采取果断措施。
```
## prominence
```
n. 突出, 重要, 卓越, 出名

例句

The website gave prominence to this news this morning.

今天早上网站在显著位置登出了这条新闻。
```
