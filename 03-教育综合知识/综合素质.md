## 职业理念
- 教育观
```
内涵：“提素个性创两全”
提高国名素质为根本宗旨——德育为先
面向全体学生的教育——所有学生“好”与“坏”
促进学生的全面发展——德智体美劳
促进学生个性发展——个体差异因材施教
培养学生的创新精神和实践能力为重点的教育——新、动手、走出去
```

- 新课改下的教学观
```
教育者为中心——学习者为中心
教会学生知识——教会学生学习
重结论轻过程——重结论更重过程
关注学科——关注人
```
- 学生观（以人为本）——“两独一发”
```
学生是发展的人：
身心发展是有规律的（循序渐进）、具有巨大的发展潜能（相信学生）、处于发展过程中人（容错）、
学生是独特的人：
完整的——不是机器人，而是有着丰富个性完整的人，不能割裂
独特性——成长环境不同，因材施教
与成人之间巨大差异——位思考
学生是具有独立意义的人：
独立于教师意志之外，不为教师意志所转移——尊重想法
学生是学习的主体——调动积极性，主动学习
权责的主体——保护合法权利，引导承担责任
```
- 学生观（全面发展）
```
涵义：智力与体力的充分发展
全面发展与个性发展不冲突，承认学生差异与个性发展为基础。面向全体学生，重视每一位学生发展。
全面发展≠均衡发展，不能用同样的标准去衡量所有方面的发展指标。
全面发展的构成：德智体美劳。
教育公正与学生共同发展：
教育公正（面对全体学生）：入学机会均等，教育过程中、取得学业成功。
学生共同发展：性别、家庭背景、地域、民族、
身心水平。
```
