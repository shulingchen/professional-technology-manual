## 语文教学设计指导思想
- 发挥师生的主动性和创造性
```
引导、指点、指向
氛围——打造良好的课堂气氛:重视学生活动
教法——激发学生学习兴趣：灵活
授法——引导学生主动学习：学习方法
```
- 努力体现语文的综合性和实践性
```
1、倡导启发式教学、讨论式教学
2、综合性：沟通课堂内外，结合家、校、社等的教育资源开展综合性学习；扩宽学生学习空间；增加学生学习语文实践的机会。
```
