## slender
```
/'slendə(r)/

adj. 苗条的, 纤细的, 窄的, 微薄的

The models on stage impressed people with their tall and slender figures.

舞台上模特们高挑又纤细的身材令人们印象深刻。
```
## stoop
```
v. 弯腰, 俯身, （站立或行走时）弓背
n. 门廊, 驼背, 曲背

The street cleaner stooped down to pick up the bottles littered all around.
道路清洁工弯下腰捡起被乱扔了一地的瓶子
```
## vigorous
```
adj. 用力的；体力的；充满活力的；强壮的；健康的；精力旺盛的

例句

Take vigorous exercise for several hours a week.

每周做几个小时的剧烈运动。
```
## prompt
```
adj. 迅速的，立刻的；敏捷的

v. （事件，事实）引起，激起；促使（某人）采取行动；提示

n. 鼓励，催促；提示；【计算机】提示信息；【给演员的】提词，提白

例句

Prompt action was required to prevent it from becoming worse.

为防止事情进一步恶化，须立即采取行动。
```
## rapidity
```
n. 迅速；速度

例句

The weather can change with amazing rapidity.

天气能以惊人的速度发生变化。
```
