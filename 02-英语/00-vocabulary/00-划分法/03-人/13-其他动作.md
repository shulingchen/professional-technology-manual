## swallow
```
/'swɒləʊ/
v. 吞下，咽下；吞没；相信，轻信
n. 咽，一次吞咽的量；燕子

Always chew food before swallowing it.

什么食物都要先嚼碎再吞咽。
```
## punch
```
n. 一拳，一击；感染力，吸引力；打孔器

v. 用拳击打；敲击（按键）；打孔
The boxer gave his opponent a punch on the nose.

这个拳击手朝他对手的鼻子上猛击一拳.
```
## sticky
```
adj. 黏（性）的, 一面带黏胶的, 闷热的, 感到热得难受的

n. 告事贴
I have to take a shower before going out because the sweat had made my skin sticky.

出门前我得冲个澡，因为汗水让我的皮肤黏乎乎的。
```
## plunge
```
v. （使）突然向前倒下[跌落]；价格、比率等〕暴跌，骤降；〔船〕猛烈地颠簸

n. 突降；俯冲；〔价格、价值等的〕骤降；突然卷入；跳水；〔短时间的〕游泳

He plunged into cold water to save the drowning girl.

他纵身跳入冰冷的水中去救这个溺水的女孩。
```

## murmur
```
v. 低声说；私下抱怨，嘀咕

n. 低语声，轻柔的谈话声；私下抱怨，嘀咕

例句

An angry murmur ran through the crowd.

群众纷纷气愤地抱怨着。
```

## grope
```
v. 摸索；搜寻；猥亵；摸（某人）身体

n. 猥亵；抚摸

例句

He groped around in the dark for his other sock.

他在黑暗中到处瞎摸找另一只袜子。
```

## crane
```
v. 伸长(脖子)以便看得更清楚; 探头看; 起吊

n. 鹤; 起重机; 吊车; 类似起重机的装置
He craned his neck above the crowd to look for his son.

他把头探出人群，寻找他的儿子。
```
## sneeze
```
v. 打喷嚏

n. 打喷嚏；喷嚏声

例句

When we sneeze, our eyes close.

打喷嚏的时候眼睛会闭上。
```
## crawl
```
vi. （人）爬行；匍匐前进

例句

Babies often crawl on the floor before they learn to walk.

婴儿在学会走路前，经常在地上爬。
```
## mount
```
v. 登上(山等); 骑上(马、车)等; 增加, 增长; 上升; 安装; 镶嵌; 裱贴; 开展; 举行

n. 供骑乘的东西; 底座, 座架; 装帧, 框; 扇托

例句

Pressure is mounting on the government to change the law.

迫使政府修改法律的压力不断增加。
```

```
例句

我赶忙去确认她是否安全。
```
