## foam
```
n. 泡沫，泡沫剂；海绵橡胶

v. 有泡沫，起泡沫

例句

He bought a foam pillow.

他买了一个海绵枕头。
```
## sponge
```
n. 海绵动物, （天然或人造、擦洗物品用的）海绵块

v. 揩拭, 用湿布（或海绵）擦, 用湿布（或海绵）擦掉, 揩去

例句

The housewife picked up a sponge to clean the edges of the sink.

这个主妇拿起一块海绵去擦拭水槽边缘。
```
## conditioner
```
n. 护发剂, 护发素;（洗衣后用的）柔顺剂;  调节剂; 改良剂；调节器

例句

This conditioner contains a nutrient that is beneficial to scalp.

这种护发素含有对头皮有益的营养成分。
```
