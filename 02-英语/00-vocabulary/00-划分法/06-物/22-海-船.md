## drift
```
v. 漂流；缓慢移动；漂泊

n. 漂流，移动，（情形，意见的）渐变，趋势

The empty boat is drifting out to sea.
这艘空船正向大海漂去。
```

## canoe
```
/kə'nuː/
n. 划艇, 独木舟

v. 划（或乘）划艇

Shall we go travelling in a canoe?

我们一起乘独木舟旅行吧？
```

## vessel
```
/'vesl/
n. 船；容器；血管

A fishing vessel sank after colliding with a cargo ship.
一艘渔船在与货船相撞之后沉没了。
```

## ashore
