## antique
```
n. 古物, 古董

adj. 古时的; 古老的; 古风的, 古式的; 古董的; 陈旧的, 过时的

例句

I saw the vase in the window of an antique shop.

我在一家古玩店的橱窗里看见了那个花瓶。
```
## precious
```
adj. 宝贵的，珍贵的；贵重的

例句

The crown was set with precious jewels—diamonds, rubies and emeralds.

王冠上镶嵌着稀世珍宝有钻石、红宝石、绿宝石。
```
