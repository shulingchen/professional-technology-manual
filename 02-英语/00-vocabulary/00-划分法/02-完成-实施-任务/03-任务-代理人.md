## assignment
```
/ə'saɪnmənt/
n. （布置的）作业；（分配的）任务

John worked on the class assignment for a whole night.
约翰写课堂作业写了一整晚。

task 指被派或自愿承担的工作，尤指困难、辛劳的工作；

job常指短期任务；

assignment指分派做的一种特定任务，也指长期性的工作


assign vt.分配 指定 （把财产，权利等）让与 N，让渡人，受让人 代理
```

## deputy
```
/'depjuti/
n. 副手；副职；代理人
adj. 代理的，副的

I'm acting as deputy till the manager returns.
我在经理回来之前代行他的职务。
```
## role
```

She refused to take on the traditional woman's role.

她拒绝承担传统妇女的角色。
```
