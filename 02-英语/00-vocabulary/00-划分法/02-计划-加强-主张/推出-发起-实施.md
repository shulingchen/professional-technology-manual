## launch
```
v. 开始从事，发起，发动（尤指有组织的活动）；（首次）上市，发行；使（船，尤指新船）下水；发射；把（航天器、武器等）发射上天；水中发射

n. （航天器的）发射；（船的）下水；（产品的）上市；（事件的）发起；大型汽艇；机动大舢板；交通艇

例句

The NGO launched a fund-raising campaign for its new project.

这家非政府组织为其新项目发起了一次筹款活动。
```
## proceed
```
v. 开始行动；开展，（行动）进行；继续；接着做

例句

We're not sure whether we should still proceed with the work.

我们不确定是否还应该继续这项工作。
```
## charter
```
n. 特许，凭照，许可证

v. 特许成立；发特许执照；给予…特权

例句

The company was granted a charter trade in the occupied territory.

这家公司被授予特权，可以在被占领地区进行贸易活动。
```
## privilege
```
n. 特惠待遇；（个人或群体的）特权；荣幸

v. 给予特权，特别优待

例句

Education should be a universal right and not a privilege.

教育应当是全民的权利而非某部分人特别享有的。
```
