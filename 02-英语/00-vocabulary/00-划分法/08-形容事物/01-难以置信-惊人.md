## incredible
```
/ɪn'kredəbl/

adj. 难以置信的，惊人的；极好的；极大的

It seemed incredible that she had been there a week already.

真让人难以置信，她已经在那里呆了一个星期了。
```

## extraordinary
```
/ɪk'strɔːdnri/

adj. 意想不到的，不平常的；非凡的；特派的；临时的

Madame Curie was an extraordinary scientist.
居里夫人是一位卓越的科学家。

It's extraordinary that he managed to sleep through the party.
真想不到他竟然从聚会开始一直睡到结束。
```
## bizarre
```
adj. 离奇的, 奇特的, 奇怪的
The clown in the circus attracted much attention by wearing bizarre costumes.

马戏团的小丑身穿奇装异服，非常吸引人们的注意。
```
## undoubtedly
```
adv. 无疑, 毋庸置疑地

Dying to Survive is undoubtedly one of the best movies of this year.

《我不是药神》无疑是今年的最佳影片之一。
```
## astonishment
```
n. 惊讶，惊异

例句

To our astonishment, they arrived on time.

使我们惊讶的是，他们准时到了。
```
## exotic
```
adj. 来自异国（尤指热带国家）的, 奇异的, 异国情调的, 异国风味的

n. 舶来品, 外来物, 外来品种, 外来语

例句

The restaurant is well-known for its delicious exotic foods.

这家餐馆因其可口的异域美食而为人熟知。
```
## fuss
```
n. （为小事）大惊小怪，小题大做；瞎忙；（具体的）反对，抗议，纷争；繁琐的手续；麻烦，困难

v. 过分关心（枝节小事）；（为小事）烦恼；打扰，烦扰

例句

Why are you making such a fuss about it?

你为什么对此大惊小怪？
``` 
## startle
```
v. 惊吓；使吓一跳

n. 惊愕；惊恐

例句

The explosion startled the children.

爆炸声让孩子们吓了一跳。
```
## marvel
```
n. 令人惊异的人（或事）；奇迹

v. 感到惊奇；大为赞叹

例句

They marveled that she would make such a decision.

他们对她做出如此决定感到惊奇。
```
