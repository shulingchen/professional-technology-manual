## sort
```
n. 排序, 分类, 种类, 类别

v. 整理, 把…分类, 妥善处理, 安排妥当

例句

This sort of problem is quite common./ These sorts of problems are quite common.

这类问题相当普遍。╱这几类问题相当普遍。   
```
## classic
```
adj. 典型的；一流的，经典的

n. 名著；优秀的典范

例句

Interest in classic music has revived recently.

近来对古典音乐的兴趣又浓厚起来。
```
## breed
```
n. （动物）品种；（人的）类型

v. （动物）繁殖；饲养，培育；孕育，导致；使养成

例句

His horse is of the best breed.

他的马是最好的品种。
```
## reproduce
```
v. 繁殖生育；再生产，再制造；复制，复写；被复制，被印制；（在不同情况下逼真地）仿造，仿制

例句

Most fish reproduce themselves by laying eggs.

大多数鱼产卵繁殖。
```
