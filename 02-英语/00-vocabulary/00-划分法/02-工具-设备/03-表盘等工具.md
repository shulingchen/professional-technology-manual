## dial
```
/'daɪəl/
n. 表盘, (钟表等的)针盘, (仪表等的)标度盘, (电话的)拨号盘

v. 拨(电话号码), 用矿用罗盘测量, 调, 用标度测量

Check the tyre pressure on the dial.

检查一下仪表盘显示的车胎压力。
```

## pedal
```
n. 踏板

adj. 踏板的；脚踏的

v. 踩踏板；骑自行车

他使劲踩下刹车踏板以避免撞车。

```

## bolt
```
n. 螺栓, 插销, 闪电, 弩箭

v. 狼吞虎咽, 用插销闩上, 能被闩上, 用螺栓把（甲和乙）固定在一起, 突然跑出

adv. 突然

例句

I closed the door and drew the bolt.

我关上门，拉上了插销。
```
## gear
```
n. 排挡；齿轮；传动装置;（某种活动的）设备，用具；衣服

v. 用齿轮连接；（使）适合，使适应；（使）准备好

adj. 时髦的；好极了

例句

Careless use of the clutch may damage the gears.

离合器使用不慎可能会损坏传动装置。
```
