## salute
```
/sə'luːt/

v. 敬礼；致敬；表示敬意
n. 敬礼；致敬；致意；鸣炮致敬

He came forward and saluted the Queen.
他站出来向女王致敬。
```

## compliment
```
/'kɒmplɪmənt/
n.恭维；敬意；道贺，贺词；致意
vt.向…道贺；称赞；向…致意；

Compliments can make one's day better.

You should take it as a compliment.
你应该把它看成是一种赞赏。
```

## gracious
```
adj.亲切的，和蔼的；

Phil is most gracious to everyone.
菲尔对每个人都很亲切。
```

## grace
```
n. 优雅，优美；风度，体面；恩典；宽限期

v. 为…增色，美化，使优美；装饰
She acts with the natural grace of a dancer.

她的动作中有舞蹈演员自然而然的优雅。
```

## gracefully
```
adv.
优美地；优雅地；雅致地；
He dived gracefully into the water.
他动作优美地跳入水中。
```
