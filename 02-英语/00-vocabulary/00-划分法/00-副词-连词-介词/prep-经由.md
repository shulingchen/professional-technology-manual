## via
```
/'vaɪə; ˈviːə/

prep. 通过, 经由
n. 道路, 【医】管道

I heard about the sale via Jane.
我从简那里听说了这次大减价。
```
## per
```
prep. 每，每一（与单位连用，表示比率）；经，由

例句

Rooms cost 50 per person, per night.

房价每人每晚50英镑。
```
## thereby
```
adv. 因此; 由此; 从而

例句

He became a citizen in 2000, thereby gaining the right to vote.

他在2000年成为公民，由此获得了投票权。
```
