## authentic
```
adj. 真正的, 真品的, 真迹的, 真实的

例句

I don't know if the painting is authentic.

我不知道这幅画是不是真迹。
```
## earnest
```
adj. 非常认真的, 诚实的, 真诚的

n. 认真, 热心, 保证金, 预兆

例句

The cold-blooded man turned down her earnest plea for help.

这个冷血的男人拒绝了她诚恳的求助。
```
## exceptional
```
adj. 优越的；杰出的，优秀的；异常的，特别的

例句

He has an exceptional memory.

他有非凡的记忆力。
```
