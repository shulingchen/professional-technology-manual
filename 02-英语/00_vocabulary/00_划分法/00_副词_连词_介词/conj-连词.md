## Whereas
conj. 鉴于；然而；反之；
n. “鉴于…”：正式文件的介绍性陈述，开场白；条件性陈述；
```
Some people like coffee, whereas others like tea.
有人喜欢咖啡,然而也有人喜欢茶。

Whereas we want a flat, they would rather live in a house.
我们想住公寓房，而他们却想住一所房子。
```

## lest
```
/lest/
conj.唯恐；以免；惟恐；生怕

They spoke in whispers lest they should be heard.

他们低声耳语唯恐被人听见。
```
