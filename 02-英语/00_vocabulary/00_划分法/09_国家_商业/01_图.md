# 图表
## illustration
```
/ˌɪlə'streɪʃn/
n.说明；例证；图解；插图；

The color illustrations in that book can draw children's attention.
书中的这些彩色插图能够吸引孩子们的注意力。
```

## diagram
n.图表；图解；示意图；[数]线图
vt.用图表示；图解；
```
The results are shown in diagram 2.

结果显示在表2上。

一般柱状图（bar chart）流程图（flow chart）饼图（pie chart）用chart表示 

diagram 表示简图，图解，图表，示意图
```
