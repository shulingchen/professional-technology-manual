## abolish
```
/ə'bɒlɪʃ/

v. 废止，革除

There are many bad customs and laws that ought to be abolished.
有许多不良的习俗和法规应予以废除
```

## restriction
```
/rɪ'strɪkʃn/

n. 限制, 约束, 制约因素, 限制规定

Drivers should be familiar with the speed restrictions.

对于限速，司机们再熟悉不过。
```

## restrict
```
/rɪ'strɪkt/

v. 限制，约束；限定

We restrict the number of students per class to 10.

我们将每个班的学生人数限定为10人。
```
