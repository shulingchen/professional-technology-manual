### 如何提交文件到服务器

1. 进到项目根目录，右键打开终端reveal in File Explorer
在输入框输入： cmd  确定


2. 查看哪些文件新增
git status

3. 添加文件
git add .

菜单栏三，Changes下+相当于add .
4. 提交到本地
git commit -m"feat: 描述信息"

5. 最后推送到6.把改动的内容拉下来
git push

6. 把改动的内容拉下来
git pull
